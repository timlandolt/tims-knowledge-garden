#shortcuts #terminal

---

## Vim
- selecting
- copy & paste
- go to start / end of line
- deleting a chunk

## Terminal
- new window
	- **⌘ N**
- New Tab
	- **⌘ T**
- split window
	- Horizontally: **⌘ D**
	- Vertically: **⌘ ⇧ D**
- Focus other window
	- **⌘ ⌥ 2**
- Focus other tab
	- Last: **⌘ \[**
	- Next: **⌘ ]**
- delete current input
	- all left: **⌃ U**
	- all right: **⌃ K**
- Move cursor
	- to start: **⌃ A**
	- to end:  **⌃ E**

## Mac
- Navigate between windows
	- Select application: **⌘ ↹**
	- Select Window: **⌘ \`**
